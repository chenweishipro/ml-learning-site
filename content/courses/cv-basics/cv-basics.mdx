# 计算机视觉基础

CV 是让机器"看懂"图像和视频的技术。深度学习时代它飞速发展, 从分类到检测到分割。

## 1. 图像表示

一张彩色图像是一个三维张量: `[H, W, 3]`, 三通道是 RGB。

像素值范围 `[0, 255]`, 训练时要归一化到 `[0, 1]` 或 `[-1, 1]`。

```python
import numpy as np
img = np.random.randint(0, 256, (224, 224, 3), dtype=np.uint8)
gray = 0.299*img[:,:,0] + 0.587*img[:,:,1] + 0.114*img[:,:,2]
```

## 2. 卷积操作

卷积核 (kernel) 在图像上滑动, 每个位置做加权求和。常见尺寸 `3x3`, `5x5`。

作用: 提取局部特征。水平边缘检测核: `[[-1,-1,-1], [0,0,0], [1,1,1]]`。

## 3. 边缘检测

经典算子: Sobel / Prewitt / Laplacian。本质是用卷积核求像素梯度。

- Sobel X: 检测垂直边缘
- Sobel Y: 检测水平边缘
- 梯度幅值: `sqrt(Gx^2 + Gy^2)`

## 4. 滤波器对比

低通滤波 (均值 / 高斯): 去噪, 模糊
高通滤波 (Sobel / Laplacian): 增强边缘, 锐化

## 5. 形态学操作

腐蚀 / 膨胀 / 开闭运算, 处理二值图像。常用于去噪小斑点、填补空洞。

## 6. 特征工程时代

传统 CV 用 SIFT / HOG / SURF 手工提取特征, 再喂 SVM。

深度学习后, 特征自动学习, 手工特征基本退场。
