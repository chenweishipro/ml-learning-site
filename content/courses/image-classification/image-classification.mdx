# 图像分类网络

图像分类是 CV 最经典的任务: 给一张图, 输出它属于哪一类。从 LeNet 到 ResNet, 网络越来越深。

## 1. LeNet-5 (1998)

第一个成功的 CNN, 5 层 (2 卷积 + 2 池化 + 1 全连接), 用在 MNIST 手写数字识别。

输入 32x32 灰度图, 输出 10 类。参数量 ~60K。

## 2. AlexNet (2012)

ImageNet 比赛冠军, 8 层, 5 卷积 + 3 全连接。引入 ReLU + Dropout + GPU 训练。

Top-5 错误率从 26% 降到 15%。深度学习时代开启。

## 3. VGGNet (2014)

VGG-16: 13 卷积 + 3 全连接, 全部 3x3 卷积堆叠。参数量 ~138M, 但结构简洁优雅。

证明: 小卷积核堆叠 > 大卷积核单层。

## 4. ResNet (2015)

核心创新: 残差连接 `y = F(x) + x`。解决深层网络梯度消失问题。

ResNet-152: 152 层, 但 Top-5 错误率仅 3.57%, 超越人类水平 (~5%)。

## 5. 演进对比

| 网络 | 年份 | 层数 | Top-5 错误 |
|------|------|------|-----------|
| LeNet | 1998 | 5 | - |
| AlexNet | 2012 | 8 | 15.3% |
| VGG-16 | 2014 | 16 | 7.3% |
| ResNet-152 | 2015 | 152 | 3.57% |

## 6. 训练技巧

数据增强: 随机裁剪 / 翻转 / 颜色抖动
正则化: Dropout / Weight Decay / BatchNorm
优化: SGD + Momentum 或 Adam, 学习率 warmup + cosine decay

## 小结

图像分类网络从浅到深、从手工到自动特征。下章讲目标检测, 不仅要分类还要定位。
